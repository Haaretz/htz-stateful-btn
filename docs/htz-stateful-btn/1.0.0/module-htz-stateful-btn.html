Parameters

The button to manage.
inStateText String <optional>

Text used as a label for accessibility purposes. Can also be alternatively provided by means of a data-in-state-text attribute on the btn element. If provided both in the html and as an argument to the function, the js argument takes precedence.
inStateClass String <optional>

Class(es) to add when enabling state. Can also be alternatively provided by means of a data-in-state-class attribute on the btn element. If provided both in the html and as an argument to the function, the js argument takes precedence.
